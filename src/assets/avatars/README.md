# Avatare

Lege deine 12 Tierbilder **direkt in diesen Ordner**. Sie werden automatisch
geladen (`index.ts` nutzt `import.meta.glob`) — kein Code-Änderung nötig.

- Erlaubte Formate: `.png`, `.jpg`, `.jpeg`, `.webp`, `.svg`
- Empfohlen: quadratisch, ~512×512, als `.webp` komprimiert (kleiner = schneller)
- Reihenfolge = alphabetisch nach Dateiname (egal, da pro Runde gemischt)

Beispiel-Namen:

```
chicken.webp  cow.webp  monkey.webp  panther.webp
gorilla.webp  hyena.webp  lion.webp   donkey.webp
cat.webp      lion2.webp  chimp.webp  rat.webp
```

Solange keine Bilder da sind, zeigt die App Emoji-Platzhalter (🐔🐮🐵…).
Sobald du Bilder einlegst, werden sie automatisch verwendet.
